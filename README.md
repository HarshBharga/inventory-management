# inventory-management

# requirment
 1. PHP 8.4
 2. Composer 2.0
 2. MySQL 8.0

## 🛠️ Installation

```bash
  composer install
  php artisan migrate
  php artisan db:seed
```


## 🛠️ API Document Import to Postman

"api-document.json" file from root directory of the project.

## 🛠️ Test

```bash
  php artisan test
```
